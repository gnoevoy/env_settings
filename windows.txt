General text editing (Notes)
------
Ctrl + Backspace -> delete the previous word
Ctrl + Delete -> delete the next word
Ctrl + End / Home -> Move the cursor to the end / beginning of the document
Ctrl + F -> Find text
Ctrl + X -> Cut (Deleting and copying to clipboard)
Ctrl + Z / Y -> Undo / Redo
End / Home -> Move the cursor to the end / beginning of the line
Shift + End / Home -> Select text from the cursor to the end / beginning of the line
Ctrl + Shift + End / Home -> Select text from the cursor to the end / beginning of the document
Tab -> Indent line 

-----------------------------------

Notebook:
------
Ctrl + N -> New notebook
Ctrl + S -> Save notebook
Ctrl + W -> Close selected page
Ctrl + O -> Open existing notebook
Ctrl + Tab -> Switch between open notebooks
Ctrl + Shift + Tab -> Switch between open notebooks in reverse order
Ctrl + Numbers -> Switch to the corresponding page in the notebook (1-9)

Alt + Tab -> Switch between open applications
Ctrl + Alt + Tab -> View a thumbnail of all open apps 

Windows + Arrows -> Move window app 
Windows + D -> Show desktop
Windows + Tab -> Open Task View
Windows + Number -> Open the corresponding app in the taskbar (1-9)
Windows + Shift + Number -> Open a new instance of the corresponding app in the taskbar (1-9)

Arrows -> Move between tabs
Delete -> Close selected tab



Alt + Shift -> Select language


Windows -> Open start menu
Ctrl + Shift + Esc -> Open Task Manager

Ctrl + Print Screen -> Full scren screenshot
Print Screen -> Make screenshot

Windows + A -> Open Action Center 
Windows + E -> Open File Explorer

Windows + L -> Lock your PC
Windows + I -> Open Settings
Windows + N -> Open Calendar
Windows + X -> Power User Menu
Windows + Q -> Open Search
Windows + . -> Open emoji panel

Windows + Shift + T -> Text Extractor
